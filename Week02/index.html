Check Boxes

Check boxes allow the user to select more than one option at once. Check boxes all have the same name but different values! The value is what is submitted by the form, not the text or picture or icon next to the checkbox.

Radio Buttons

Radio fields allow the user to select ONLY one option at once. Radio fields have the same name but different values! The value is what is submitted by the form, not the text or picture or icon next to the checkbox.
