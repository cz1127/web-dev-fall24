Relative
Absolute positioned divs look for a parent element with a defined position property to position against. If it can't find one, it positions absolutely to the body tag.
Fixed behaves similarly to absolute but will be fixed on the screen regardless of scrolling

when we have a relative, parent, absolute or fixes position child elements will position in relation to the relativel positioned parent
